# service booking website
A platform where customers can find and book service providers (hair cutting, car detailing, etc.).
## Project structure
- `frontend/` - React Native (web + mobile)
- `backend/` - Node.js API server
- `shared/` - Shared TypeScript types
- `docs/` - Project documentation

## Tech Stack
- Frontend: React Native (web + iOS/Android)
- Backend: FastAPI, Python
- Database: PostgreSQL

## Development Phases
1. ✅ Project Setup & GitHub Repository
2. ⏳ User Authentication & Basic Profiles
3. ⏳ Service Catalog & Search
4. ⏳ Booking System
5. ⏳ Payment Processing
